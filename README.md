## Домашнее задание №1

1 [2] Программа должна уметь стартовать и убивать окружения с помощью команд:
 - a [1] стартовать окружения - ./main.sh start N
 - b [0.5] убивать одно конкретное окружение - ./main.sh stop i
 - c [0.5] убивать все окружения с помощью ./main.sh stop_all

2 [1] Каждое созданное окружение должно жить в своем tmux-окне (window)

3 [0.5] При старте окружений должен выводиться сообщение в stdout об этом

4 [1.5] У каждого окружения должны быть:
 - a [0.5] отдельная рабочая директория + (для прод.) отдельное виртуальное venv окружение
 - b [0.5] отдельный сетевой порт
 - c [0.5] отдельный (уникальный и случайный) токен
